# Carcy-BAP



### Installation

Récuperation du git
```
git clone 

```
Installation du projet
```
npm install
```
 Fixtures
```
symfony fixtures:load
```
Migré la base de donner
```
symfony make:migration
```


### Excuté
```
symfony doctrine:migration:migrate
```
### Lancer le serveur 
```
symfony server:start

```
### Stop le serveur
```
symfony server:stop
```
